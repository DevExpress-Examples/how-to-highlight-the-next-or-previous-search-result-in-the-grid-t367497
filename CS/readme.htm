Developer Express Example
How to highlight the next or previous search result in the grid

This example demonstrates how to implement a search functionality like in Opera using the ButtonEdit (https://documentation.devexpress.com/#WindowsForms/clsDevExpressXtraEditorsButtonEdittopic)editor. To use this solution in your application, execute the following steps: 1. Drop the FindHelper component onto the target form. 2. Assign a GridControl to the FindHelper.TargetControl property. 3. Assign a required ButtonEdit to the helper's SearchControl property. See also: ButtonEdit (https://documentation.devexpress.com/#WindowsForms/clsDevExpressXtraEditorsButtonEdittopic) GridControl (https://documentation.devexpress.com/#WindowsForms/CustomDocument3461) http://www.devexpress.com/scid=T368346
